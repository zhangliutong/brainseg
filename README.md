# brainseg
This is the source code for our paper "One-shot Neuroanatomy Segmentation through Online Data Augmentation and
Confidence Aware Pseudo Label"

# Requirements
- pytorch >= 1.6.0
- python >= 3.7
- Some basic python packages such as Numpy, nibabel

# Usage
1. Train the model  
python train.py  
3. Test the model  
python test.py
